== Description ==

Noor Responsive Tab is ultimate wordpress tab plugin. You can customize lot of things such as tab title color, tab fonts with font-awesome icons, tab width, tab background, tab active color, tab hover color. Its support categories so you can easily create multiple tabs with multiple settings.

== Installation ==

Installing this plugin as regular wordpress plugin.

After install, this plugin enable a custom post called <strong>Tabs</strong>. Add category & items on that custom post. 
After adding items, you are ready to use this plugin in your post, page, widget or anywhere you like. 

Go to page > Add New. On the editor, you will see a button called Tab Shortcode by clicking button you see the shortcode like <code>[rstab category="Category_Name"]</code>

Use category title, not category id or slag. Make sure spelling is ok. It is case sensitive.
